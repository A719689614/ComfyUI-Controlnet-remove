# ComfyUI-Controlnet-remove
ComfyUI 控制网格移植版(快捷键)
# 一、功能介绍
其实和官方的controlnet一样，我做了美化和快捷方式，主要输入相对应的数字前缀就可以调用！

然后我做了文件夹索引嵌套，是不支持文件夹改名字的，你下载到custom node底下解压即可！
